Sets up a subscription for the information about value changes of defined single variable on the OPC UA client. The OPC UA server monitors the single variable and sends notification when a value changes. Retrieves monitored single variable (monitored item).

This function block works asynchronously and must therefore be called while Active == TRUE.

After a task is transferred, error 16#80040000 (BadResourceUnavailable) is output for the variable on input "Variable". Therefore the function block must be disabled and enabled again. To achieve this is recommended to disable the function block in the exit routine.

To disable this function block it must be called until Active == FALSE.
