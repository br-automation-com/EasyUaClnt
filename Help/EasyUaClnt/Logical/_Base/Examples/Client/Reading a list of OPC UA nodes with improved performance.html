Reading a list of OPC UA nodes with improved performance

The OPC UA server nodes to read are VarX, VarY and VarZ and the value will be taken from the local variable VarA, VarB and VarC. The code in the server side regarding VarX, VarY and VarZ is: 

		PROGRAM _CYCLIC
		    VarX := VarX +1;
		    VarY := VarY +1;
		    VarZ := VarZ +1;
		END_PROGRAM

Reading the nodes can be done executing the FUB EasyUaReadBulk_0 by setting EasyUaReadBulk_0.Execute to TRUE. This can be done from the Watch window of the ReadBulk task.
