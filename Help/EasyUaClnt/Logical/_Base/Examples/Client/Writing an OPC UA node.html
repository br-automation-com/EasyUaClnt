Reading the OPC UA node

The OPC UA server node to write is VarX and the value will be written in the local variable VarA. The code in the server side regarding VarX is: 

		PROGRAM _CYCLIC
		    VarX := VarX +1;
		END_PROGRAM

Reading the node can be done executing the FUB EasyUaWrite_0 by setting EasyUaWrite_0.Execute to TRUE. This can be done from the Watch window of the Write task.
