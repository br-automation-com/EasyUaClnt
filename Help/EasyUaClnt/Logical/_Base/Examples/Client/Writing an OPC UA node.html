Writing an OPC UA Node Using EasyUaWrite

Program examples are developed on ARsim, B&R's simulation PLC.

Preparing the OPC UA Client

The OPC UA client is enabled in the Configuration View, under the active configuration, in the following file: Connectivity > OpcUaCs > UaCsConfig.uacfg.

Parameter "OPC UA Client/Server" is set to "Enabled". The parameter TCP port number is set to 4841.

Adding EasyUaClt Library

In order to run the sample, EasyUaClt has to be imported using Toolbox > Existing Library.

Sample Program on the Client

Program example "Write" runs on the PLC implementing the FUB EasyUaWrite. It can be saved using the following link. Then it has to be imported in the client Automation Studio project.

Write Sample Task

Reading the OPC UA Node

The OPC UA server node to write is VarX and the value will be written in the local variable VarA. The code on the server side regarding VarX is:

PROGRAM _CYCLIC
    VarX := VarX + 1;
END_PROGRAM

Reading the node can be done by executing the FUB EasyUaWrite_0 by setting EasyUaWrite_0.Execute to TRUE. This can be done from the Watch window of the Write task.
