Writing a List of OPC UA Nodes Using EasyUaWriteBulk

Program examples are developed on ARsim, B&R's simulation PLC.

Preparing the OPC UA Client

The OPC UA client is enabled in the Configuration View, under the active configuration, in the following file: Connectivity > OpcUaCs > UaCsConfig.uacfg.

Parameter "OPC UA Client/Server" is set to "Enabled". The parameter TCP port number is set to 4841.

Adding EasyUaClt Library

In order to run the sample, EasyUaClt has to be imported using Toolbox > Existing Library.

Sample Program on the Client

Program example "WriteBulk" runs on the PLC implementing the FUB EasyUaWriteBulk. It can be saved using the following link. Then it has to be imported in the client Automation Studio project.

WriteBulk Sample Task

Writing a List of OPC UA Nodes with Improved Performance

The OPC UA server nodes to write are VarX, VarY, and VarZ, and the value will be taken from the local variable VarA, VarB, and VarC. The code on the server side regarding VarX, VarY, and VarZ is:

PROGRAM _CYCLIC
	VarX := VarX + 1;
	VarY := VarY + 1;
	VarZ := VarZ + 1;
END_PROGRAM

Writing the nodes can be done by executing the FUB EasyUaWriteBulk_0 by setting EasyUaWriteBulk_0.Execute to TRUE. This can be done from the Watch window of the WriteBulk task.
